# online_video_player

ローカルの動画ファイルをブラウザで再生する単一ページの動画プレイヤーです。

## 機能

- 複数動画の読み込み
- 再生 / 一時停止
- 前後の動画への移動
- 再生速度の変更
- ドラッグ&ドロップによる動画読み込み
- キーボードショートカット

## 使い方

1. `index.html` を最新の主要ブラウザで開く
2. `動画ファイルを選択` からローカル動画を選ぶ
3. 一覧から動画を選択して再生する

ドラッグ&ドロップでも動画を読み込めます。

## 必要環境

- 最新の主要ブラウザ
- JavaScript が有効であること
- ローカルファイルにアクセスできること

## 注意

- 動画ファイルは端末内でのみ扱われます
- 外部サービスへの送信は行いません

## ショートカットキー

- `Space`,`k`,`o`: 再生 / 一時停止
- `←`: 5秒巻き戻し
- `→`: 5秒早送り
- `P`: 1%巻き戻し
- `I`: 1%早送り
- `J`: 10秒巻き戻し
- `L`: 10秒早送り
- `↑`: 音量を上げる
- `↓`: 音量を下げる
- `M`: ミュート切り替え
- `F`: 全画面表示の切り替え
- `return`: 次の動画へ移動
- `backspace`: 前の動画へ移動
- `[` : 再生速度を下げる
- `]`: 再生速度を上げる
- `0` から `9`: 再生位置を 0% から 90% にジャンプする
- `Esc`: 全画面表示を解除する
- `.`: フレーム送り
- `,`: フレーム戻し
- `u`: ループ・自動再生モード変更
